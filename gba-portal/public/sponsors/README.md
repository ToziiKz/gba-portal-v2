# Assets sponsors

Ce dossier contient des assets statiques utilisés par la page **/sponsors**.

- `logo-placeholder.svg` : logo de démonstration (placeholder)

## Remplacer par des logos réels

1. Ajoutez les fichiers (SVG/PNG) dans `public/sponsors/`
2. Mettez à jour `src/app/sponsors/page.tsx` (tableau `logoPlaceholders`) pour pointer vers vos logos.

> Conseil : privilégiez le SVG ou le PNG avec fond transparent.
